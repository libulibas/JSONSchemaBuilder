# JSONSchemaBuilder

Turns a plain English data-structure description into a valid JSON Schema.

## Installation

```bash
pip install -r requirements.txt
cp .env.example .env  # add your ANTHROPIC_API_KEY
```

## Usage

```bash
python json_schema_builder.py "describe your structure"
```

## Model

`claude-sonnet-4-20250514` via the Anthropic Python SDK.
